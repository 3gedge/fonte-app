# Fonte marketing site

The marketing website for Fonte (https://fonte.finance), a non-custodial crypto portfolio manager on Base. Static site (HTML, CSS, vanilla JS), deployed on Cloudflare Pages.

## Pages
- `index.html`: landing page
- `business.html`: for funds (white-label and API)
- `app.html`, `app.js`: sample dashboard demo
- `platform.html`: white-label console prototype
- `about.html`, `terms.html`, `privacy.html`, `contact.html`
- `styles.css`: shared styles
- `assets/`: Open Graph images
- `.ogbuild/`: Open Graph image generation tooling

No backend and no secrets. This repository is the public marketing front end only.
